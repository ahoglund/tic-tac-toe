# tic-tac-toe

Game Play

 Run this program from the command line.
 
  # ruby tic-tac-toe.rb
  
  It will display a new tic-tac-toe game board and a prompt for the 1st player to
  enter the index # corresponding to the board position.

      1 | 2 | 3
      ---------
      4 | 5 | 6
      ---------
      7 | 8 | 9
